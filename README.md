# Vim Configuration

## Installation

```sh
~/.vim/bundle/YouCompleteMe
python3 install.py --all
```

```sh
mv ~/.ycm_extra_conf.py ~/.vim/bundle/YouCompleteMe
```

## Change Syntax Highlighting Colors
```sh
cp -r ./jellybeans.vim ./vim-cpp-enhanced-highlight ~/.vim/bundle/
```
